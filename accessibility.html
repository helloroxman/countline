Accessibility

Countline is designed to be simple, clear, and accessible.

The app supports:

○  VoiceOver labels for key controls and actions
○  Dynamic Type where possible
○  Dark Mode
○  Sufficient color contrast
○  Reduce Motion support for motion-sensitive users
○  Clear layouts without unnecessary visual noise

Some visual theme colors may vary, but Countline is designed so important information is not communicated by color alone.
